Customising interactor

So now that you have interactor implemented on your site, how do you indicate which elements you want data to be saved about?

How to specify which elements to collect data from

Interactor uses the "interaction" class to allow you to specify which elements on your page should have data collected about them.
Simply set the class attribute of the element to "interaction", like so:

<p class="interaction>

Any elements of this class will have data collected about them.

How to specify what interactions should be logged

Interactor uses a script of parameters to allow you to specify which interactions should be logged with your chosen elements.

You may remember this script from the setup, where we used it to specify the address of the server the data should be sent to. At the end of the setup, the script looked something like this:

<script>
var interactions = new Interactor({
endpoint : '127.0.0.1/postmethod'
});
</script>

By default, interactor tracks the mouseup and touchend events on any elements it is tracking. If you want information on HTML events you can find some here.
You can add events that you want to be tracked by adding them to the script above in the interactionEvents array.

For example, if I wanted to track the mousedown event on my site, I would modify the script like so:

<script>
var interactions = new Interactor({
endpoint : '127.0.0.1/postmethod',
interactionEvents : ["mousedown", "mouseup","touchend"]
});
</script>

Note that I also add the mouseup and touchend events to this array - interactor only tracks these events if no other events are specified, and if any other events are specified, interactor will exlusively track those instead.
